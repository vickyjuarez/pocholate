ESTRUCTURA BÁSICA DE HTML

La estructura básica de un documento HTML se compone de varias etiquetas principales. Comienza con <!DOCTYPE html> que indica que se trata de un documento HTML5. Luego, la etiqueta <html> encierra todo el contenido. Dentro de ella se encuentran dos secciones: <head>, donde se definen metadatos, título y enlaces a hojas de estilo; y <body>, que contiene el contenido visible de la página como textos, imágenes, tablas y enlaces. Esta estructura asegura que el navegador pueda interpretar y mostrar correctamente la información.
