Etiquetas HTML No Básicas

Estas etiquetas ofrecen funcionalidades especiales en HTML. Algunas están en desuso, pero se incluyen aquí con fines educativos.

1. <iframe>

Incrusta otra página web dentro de la actual.

2. <script>

Permite insertar código JavaScript directamente en el documento.

3. <style>

Define estilos CSS dentro del documento HTML. Aunque no se usa CSS aquí, se muestra la estructura:

4. <div>

Contenedor genérico para agrupar contenido.

Este texto está dentro de un <div>.

5. <embed>

Incrusta contenido multimedia o interactivo como PDF o video.

6. <applet> (obsoleta)

Usada para incrustar aplicaciones Java. Ya no se recomienda su uso.

<applet code="MiApplet.class" width="300" height="200"></applet>

7. <frame> y <frameset> (obsoletas)

Dividen la ventana del navegador en secciones. Reemplazadas por <iframe>.

<frameset cols="50%,50%">
  <frame src="pagina1.html">
  <frame src="pagina2.html">
</frameset>

8. <marquee> (obsoleta)

Texto en movimiento. No estándar y desaconsejada.

Texto desplazándose horizontalmente

Nota final

Las etiquetas <applet>, <frame>, <frameset> y <marquee> están obsoletas y no deben usarse en proyectos modernos. Se incluyen aquí solo con fines educativos.
