¿Qué necesita una etiqueta HTML?

Una etiqueta HTML es la unidad básica que define la estructura y el contenido de una página web. A continuación se explican sus componentes esenciales:

🔹 1. Signos angulares

Las etiquetas comienzan con < y terminan con >. Esto indica al navegador que se trata de una instrucción HTML.

🔹 2. Nombre de la etiqueta

Es el tipo de elemento que se está creando, como p para párrafo, img para imagen, a para enlace, etc.

🔹 3. Atributos (opcionales)

Son propiedades que modifican el comportamiento o apariencia del elemento. Se escriben dentro de la etiqueta de apertura.

  • href: para enlaces
  • src: para imágenes o medios
  • alt: texto alternativo
  • width, height: dimensiones

🔹 4. Etiqueta de apertura

Marca el inicio del elemento. Incluye el nombre y los atributos si los hay.

Ejemplo: <a href="https://ejemplo.com">

🔹 5. Contenido interno (si aplica)

Es el texto o elementos que están dentro de la etiqueta. No todas las etiquetas lo tienen (por ejemplo, <img> es vacía).

🔹 6. Etiqueta de cierre

Marca el final del elemento. Se escribe igual que la de apertura pero con una barra /.

Ejemplo: </a>

🧪 Ejemplo completo desglosado

<a href="https://ejemplo.com" target="_blank">Visita Ejemplo</a>
  • <a: Inicio de la etiqueta
  • href="...": Atributo con la URL
  • target="_blank": Atributo para abrir en nueva pestaña
  • >: Cierre de la etiqueta de apertura
  • Visita Ejemplo: Contenido interno
  • </a>: Etiqueta de cierre
