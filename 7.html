Hipervínculos en HTML

Un hipervínculo permite enlazar una página web con otra. Se crea usando la etiqueta <a>.

Componentes principales:

  • <a>: Etiqueta de apertura del enlace.
  • href: Atributo que indica la dirección del destino.
  • Texto visible: Lo que el usuario ve y puede hacer clic.
  • </a>: Etiqueta de cierre.

Ejemplo:

Código:

<a href="https://www.ejemplo.com">Visita Ejemplo.com</a>

Resultado:

Visita Ejemplo.com
