Etiqueta <img> en HTML

La etiqueta <img> se utiliza para insertar imágenes en una página web. Es una etiqueta vacía, lo que significa que no tiene contenido entre etiquetas de apertura y cierre.

Componentes principales:

  • src: Especifica la ruta del archivo de imagen (puede ser local o URL).
  • alt: Texto alternativo que se muestra si la imagen no se carga. También mejora la accesibilidad.
  • width: Define el ancho de la imagen (en píxeles o porcentaje).
  • height: Define la altura de la imagen.
  • title: Texto que aparece al pasar el cursor sobre la imagen.

Ejemplo básico:

<img src="paisaje.jpg" alt="Paisaje natural" width="400" height="300">

Ejemplo con atributos adicionales:

<img src="https://example.com/logo.png" alt="Logo de la empresa" title="Nuestro logo" width="200">
