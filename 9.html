Etiqueta <audio> en HTML

La etiqueta <audio> permite incrustar contenido de audio en una página web. Es compatible con formatos como MP3, WAV y OGG.

Componentes principales:

  • <audio>: Etiqueta contenedora del reproductor de audio.
  • src: Atributo que indica la ruta del archivo de audio.
  • controls: Atributo que muestra los controles de reproducción (play, pausa, volumen, etc.).
  • autoplay: Reproduce el audio automáticamente al cargar la página (opcional).
  • loop: Reproduce el audio en bucle (opcional).
  • preload: Indica si el archivo debe cargarse al cargar la página. Valores: auto, metadata, none.
  • <source>: Permite incluir múltiples formatos para compatibilidad entre navegadores.

Ejemplo básico:

Este ejemplo muestra cómo insertar un archivo de audio con controles:

<audio src="cancion.mp3" controls>
  Tu navegador no soporta el elemento de audio.
</audio>

Ejemplo con múltiples fuentes:

Ideal para asegurar compatibilidad con distintos navegadores:

<audio controls preload="auto">
  <source src="audio.mp3" type="audio/mpeg">
  <source src="audio.ogg" type="audio/ogg">
  Tu navegador no soporta el elemento de audio.
</audio>
