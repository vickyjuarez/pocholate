Etiqueta <video> en HTML

La etiqueta <video> permite insertar contenido de video en una página web. Es compatible con formatos como MP4, WebM y OGG.

Componentes principales:

  • <video>: Contenedor del reproductor de video.
  • src: Ruta del archivo de video (puede omitirse si se usan <source>).
  • controls: Muestra los controles de reproducción (play, pausa, volumen, etc.).
  • autoplay: Reproduce el video automáticamente al cargar la página.
  • loop: Reproduce el video en bucle.
  • muted: Inicia el video sin sonido.
  • poster: Imagen que se muestra antes de reproducir el video.
  • width / height: Define el tamaño del reproductor.
  • <source>: Permite incluir múltiples formatos para compatibilidad entre navegadores.

Ejemplo básico:

<video src="video.mp4" controls width="640" height="360">
  Tu navegador no soporta el elemento de video.
</video>

Ejemplo con múltiples fuentes y atributos:

<video controls autoplay loop muted poster="imagen.jpg" width="640" height="360">
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
  Tu navegador no soporta el elemento de video.
</video>
